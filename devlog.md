# บันทึกการพัฒนา

## 2026-04-05
- ปรับหน้าแรกให้มีการเลือกบทบาทอย่างชัดเจน เพื่อให้ผู้ใช้เข้าสู่ระบบตามบทบาทได้ถูกต้อง
- ปรับการแสดงปุ่มสำหรับครูและผู้บริหารให้ใช้งานง่าย พร้อมลูกเล่นเลื่อนไหลดูทันสมัย
- ออกแบบพื้นหลังหน้าแรกใหม่ให้ดูเป็นมืออาชีพและมีเอกลักษณ์มากขึ้น
- แยกสไตล์ของหน้าแรกออกจากไฟล์หลัก เพื่อให้ง่ายต่อการดูแลและปรับปรุงในอนาคต
- ล้าง CSS เก่าที่ไม่ได้ใช้ออกเพื่อให้ระบบสะอาดและไม่ซ้ำซ้อน
- เสริมความเป็น Progressive Web App (PWA) ด้วยการอัปเดต manifest, service worker และไอคอน
- เพิ่มชุดไอคอนสำหรับ PWA ในขนาดมาตรฐาน
- ปรับหน้าแรกให้เชื่อมต่อ PWA ได้อย่างถูกต้องเมื่อเปิดบนอุปกรณ์ต่างๆ
- ปรับระบบนำทางให้รองรับเวอร์ชันใหม่ของ React Router
- แก้ไขหน้า test ให้โหลดได้ปกติ ไม่มี error

## 2026-04-06
- ตรวจสอบโครงสร้างโปรเจกต์ทั้งหมดเพื่อให้การพัฒนาเป็นระบบ
- วางแนวทางการจัดไฟล์ตามมาตรฐาน React SPA
- สรุปแนวทางการจัดเรียงไฟล์และการย้ายโฟลเดอร์ให้เหมาะสมกับการขยายงาน

## 2026-04-07
- สรุป Dev Log ลงใน `devlog.md` และแสดงในเมนู About ของหน้า Settings
- ปรับแสดงผล Dev Log ให้เป็นปฏิทินเลือกวันที่ ลักษณะการ์ดเล็กลง และใช้สีสันทันสมัย
- เปลี่ยนวันที่ในการ์ดเป็นรูปแบบย่อเพื่อรองรับจำนวนวันในอนาคต
- แก้ให้หน้า Dev Log ที่แสดงใน About อ่านง่ายขึ้น โดยแยกส่วนหัวข้อและรายละเอียดชัดเจน
- ปรับโครงสร้างบทบาทให้เข้าใจง่าย: `admin` เข้าถึงทุกโหมด, `executive` เข้าถึงเฉพาะโหมดผู้บริหาร
- แยกหน้าบทบาทออกจากหน้าของแต่ละแอป เพื่อให้ระบบจัดการและขยายงานสะดวก
- รวม route ของแอปและบทบาทแบบเป็นระบบ และสร้าง guard ควบคุมการเข้าถึงตาม role
- สร้างเอกสารสรุปใน `COPILOT_MEMO.md` เพื่อให้ Copilot ใช้เป็นแนวทางสรุป dev log ได้ต่อเนื่อง
